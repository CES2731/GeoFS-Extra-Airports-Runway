# GeoFS Extra Airports Runway / Addon GeoFS额外跑道插件

## How to use

1.Copy the **main.js** and paste it into the browser console

2.Reload the terrain (**not the game!**) ,you can choose anouther airport far away from which airport you currently at,and then change back

**If the runways load successfully,you will see some message like this:**

## Add ILS data

Copy the **ilsdataloader.js** and paste it into the browser console

## About this addon

This addon add some runways which were missing in GeoFS's runways database,

the new added runways have **usable PAPI light** ,**usebale ILS** and **runway overlay**

**If you have any runways need to add,please contact me and send me the relevant materials**
